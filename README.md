# Phishing-Kit-Yt-api
Youtube Api and Openphish Feed

## Start

1. Create and activate virtual environment:

    `python -m venv venv`

2. Install packages:

    `pip install -r requirements.txt`

3. Create config file:

    `configs.json`

4. Run App
    
    `$ python phishing.py <url>`
    `$ python youtube.py`
